## 简介
本项目是生理数据大模型，实现心理健康状态的检查与评估。通过腕表等可穿戴设备测量的人们生理信号的多模态数据，来微调LLM解读数据中分析出的情绪和焦虑测评结果，帮助大众了解自我情绪和认识内在需求。


## 数据集
清华大学心理学系团队发布面向情绪研究的日常动态心理生理记录数据集DAPPER
